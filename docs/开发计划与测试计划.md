## 二、已实现功能清单

### 页面 (6 个)
- `/` 仪表盘 — 统计卡片、快速创建项目、知识库管理、项目列表
- `/projects/[id]` 项目聊天 — AI 对话、代码生成（文件标签页）、模型切换
- `/kb` 知识库管理 — 文档上传、索引列表、播种/清空
- `/kb/test` 检索测试 — 查询输入、Top-K 滑块、相似度可视化
- `/settings` 系统设置 — 服务状态、AI 统计、快捷操作
- `/dashboard` 重定向到首页

### API (9 个)
- `/api/chat` — AI 聊天（代码生成 + RAG 增强 + 模型降级）
- `/api/workflow` — Agent 工作流
- `/api/models` — 模型列表
- `/api/health` — 健康检查
- `/api/download` — ZIP 下载
- `/api/admin/seed` — 默认知识库播种
- `/api/rag/upload` — 文档上传索引
- `/api/rag/test` — 检索测试
- `/api/rag/test-seed` — 测试知识库播种
- `/api/projects/[id]/download` — 项目代码下载

### 基础设施
- ✅ React 19 + Next.js 16 App Router + Turbopack
- ✅ TailwindCSS v4 + TypeScript 5
- ✅ Vercel AI SDK v6 (streamText, ToolLoopAgent, embed)
- ✅ DeepSeek Flash/Pro + Ollama + OpenAI 多模型
- ✅ Neon Postgres + Drizzle ORM (projects, messages 表)
- ✅ Pinecone / 本地内存向量库自动降级
- ✅ 自定义 Toast/Confirm 通知系统
- ✅ Barrel exports 工程化
- ✅ 速率限制 proxy.ts + 安全头
- ✅ Vercel 部署配置

## 三、待完成功能（按优先级排序）

### P0: 核心缺失（必须完成）

#### 1. Agent 工作流可视化 (模块4)
**当前问题**: Agent 代码骨架存在但前端无法触发和查看工作流
**需要做的**:
- 在聊天界面添加「Agent 模式」切换开关
- 完善 WorkflowViewer 组件，接入实时 Agent 步骤流
- 实现工作流模板选择（生成全栈应用 / 添加功能 / 修复 Bug / 代码审查）
- Agent 执行步骤的实时可视化（思考 → 工具调用 → 结果）
- 每个步骤可展开查看输入/输出详情

#### 2. 代码生成改进 (模块2)
**当前问题**: 代码只是展示在对话框中，不能直接运行
**需要做的**:
- 生成的代码写入磁盘后自动触发 Next.js 热更新预览
- 添加 iframe 实时预览生成的应用
- 增量修改支持（Diff Patching），不每次重写全部文件
- 代码语法错误自动检测和修复循环
- 「AI Generated」标记注入（文件头注释）

#### 3. 用户认证与多租户 (文档扩展项)
**当前问题**: 没有用户系统，所有数据共享
**需要做的**:
- 集成 Clerk 或 NextAuth 实现用户登录
- 每个用户隔离的项目空间
- 权限控制（管理员/普通用户）

### P1: 体验完善（应该完成）

#### 4. RAG 检索增强 (模块3)
**当前问题**: RAG 已有基础功能但不够完善
**需要做的**:
- 混合检索（BM25 关键词 + 向量语义 + RRF 重排序）
- 检索结果高亮匹配关键词
- 知识库自动更新（代码生成后自动索引）
- 引用来源在聊天中的可视化展示（Citation 角标 + 抽屉）
- RAG 准确率统计面板

#### 5. 部署流水线 (模块5)
**当前问题**: vercel.json 存在但未配置 CI/CD
**需要做的**:
- GitHub Actions 工作流配置
- 自动化测试集成
- 预览部署（每个 PR 自动创建预览环境）
- 环境变量管理（Vercel dashboard）

#### 6. 可观测性 (模块7)
**当前问题**: 有基础日志和遥测但缺少可视化
**需要做的**:
- 在 /settings 页面添加 AI 调用图表
- Token 消耗趋势图
- 错误率监控面板
- 慢请求追踪

### P2: 扩展功能（可选的）

#### 7. Tauri 桌面端 (模块6)
- Tauri 2.0 工程创建
- Next.js 静态导出集成
- 本地模型托管（Ollama 进程管理）
- Windows 安装包构建

#### 8. PWA 离线支持
- Service Worker 注册
- 离线缓存策略
- 安装提示

#### 9. 设计稿转代码
- 集成 Figma API
- 自动转换 React 组件
